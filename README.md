# 聊天应用程序

这是一个使用PHP构建的简单聊天应用程序。它允许用户注册、登录并在聊天室中交流。该应用程序旨在演示基本的Web开发概念，包括用户身份验证和实时消息传递。
- 演示网站[点击这里](http://conexus.oeza.top/chatchat)
- 欢迎各位一起参与不断完善！共创联系QQ908756682
- 当前计划：新增允许用户自主发送图片，热门聊天室列表
